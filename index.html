저의 이야기

안녕하세요! 탄탄한 보안 지식을 바탕으로 견고한 백엔드를 구축하는 개발자, 최준형입니다. 현재 조선대학교 정보보안학과에 재학하며, 시스템의 안정성과 데이터 무결성을 최우선으로 고려하는 백엔드 개발에 깊은 흥미를 느끼고 있습니다.

저는 이론에서 멈추지 않고, 학습한 객체지향 프로그래밍(OOP) 원칙과 데이터베이스 설계 지식을 직접 코드로 구현하며 실무 역량을 다지고 있습니다. 특히, Spring Boot 프레임워크를 활용한 백엔드 개발에 집중하며, RESTful API 설계 및 구현 능력을 키우고 있습니다.

아직 배우는 단계에 있지만, 새로운 기술을 스펀지처럼 흡수하고 문제를 끝까지 파고들어 해결하는 끈기를 가지고 있습니다.

현재는 GitHub로 코드를 관리하고 익숙해지고 있으며, 스스로 '만들고, 읽고, 수정하고, 삭제하는' 기본적인 웹 기능(CRUD)을 구현하는 작은 프로젝트들을 진행하며 실제 서비스 개발의 기초를 쌓고 있습니다. 또한, 알고리즘 문제 해결 사이트를 꾸준히 풀면서 문제를 해결하는 능력과 효율적으로 코드를 짜는 습관을 기르고 있습니다.

새로운 기술을 배우고 적용하는 것을 즐기며, 동료들과 함께 시너지를 창출하고 성장하는 개발자가 되기 위해 온라인 강의 수강 및 개발 블로그 학습을 통해 꾸준히 지식을 확장하고 있습니다. 앞으로는 외부 개발 커뮤니티 활동을 통해 더 넓은 경험을 쌓을 계획입니다.

저의 개발 철학

근본적인 문제 해결

눈앞의 버그 수정뿐 아니라, 문제의 본질을 파악하여 더 견고하고 확장 가능한 코드를 작성하는 것을 목표로 합니다.

신뢰할 수 있는 시스템

사용자가 믿고 사용할 수 있는 안정적이고 견고한 백엔드 시스템을 구축하는 것을 중요하게 생각합니다. 보안 취약점을 최소화하고 데이터 무결성을 보장합니다.

실천하는 지속 학습

새로운 기술 트렌드를 주시하고, 배운 것을 실제 코드에 적용하며 지식으로 만듭니다. 끊임없이 배우고 시도하며, 어제보다 나은 개발자가 되기 위해 노력합니다.

지속적인 성장

변화하는 기술 환경에 맞춰 끊임없이 배우고 발전합니다. 어제보다 나은 개발자가 되기 위해 작은 성공들을 통해 꾸준히 성장해 나가는 것을 중요하게 생각합니다.

사용자 중심 사고

개발하는 서비스가 최종 사용자에게 어떤 가치를 제공할지 항상 고민합니다. 기술적인 만족을 넘어, 사용자의 니즈를 충족시키는 솔루션을 만들고자 합니다.

테스트 기반 개발

코드의 안정성과 신뢰성을 확보하기 위해 테스트 코드 작성을 중요하게 생각합니다. 견고한 소프트웨어를 위한 품질 검증 과정을 소홀히 하지 않습니다.
